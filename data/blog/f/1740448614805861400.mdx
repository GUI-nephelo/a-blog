### 功能特点

- **实时显示**：能够实时更新并显示目标的位置和运动轨迹。
- **多目标跟踪**：支持同时跟踪多个目标的运动路径。
- **自适应缩放**：根据显示区域自动调整比例，确保最佳视图效果。
- **历史轨迹记录**：记录目标的历史轨迹，便于后续分析和回放。

### 技术亮点

- **高效的算法**：采用优化的算法，确保在高帧率下运行流畅。
- **多线程处理**：通过多线程技术，实现数据处理与显示的分离，提升整体性能。
- **低资源消耗**：优化的代码和数据结构，减少了对硬件资源的需求。

### 应用场景

- **无人机监控**：实时显示无人机的飞行路径和目标跟踪。
- **移动目标跟踪**：用于安防、交通监控等领域，跟踪移动目标的轨迹。
- **游戏开发**：在游戏中实时显示角色或物体的运动轨迹，增强游戏体验。

### 优势

- **高效性**：实时更新和显示，延迟低。
- **可扩展性**：支持多目标跟踪，适用于各种应用场景。
- **低资源消耗**：优化的算法和数据结构，减少硬件资源占用。

### 缺点

- **硬件要求**：在高分辨率或大规模显示下，可能需要较高的硬件配置。
- **复杂场景处理**：在极其复杂的场景下，可能会出现性能瓶颈。

### 代码实现

```python
import numpy as np
import matplotlib.pyplot as plt

class TrajectoryTracker:
    def __init__(self):
        self.trajectories = []
        self.figure, self.ax = plt.subplots()
        self.scatter = self.ax.scatter([], [], s=20, c='red')
        plt.ion()
        plt.show(block=False)

    def update(self, data):
        self.trajectories.append(data)
        self.scatter.set_offsets(np.array(self.trajectories))
        plt.draw()
        plt.pause(0.001)

    def close(self):
        plt.ioff()
        plt.close()

# 示例使用
tracker = TrajectoryTracker()
for i in range(100):
    data = np.random.rand(10, 2)  # 随机生成10个点的数据
    tracker.update(data)
tracker.close()
```

### 注意事项

- 合理优化数据结构和算法，减少不必要的计算。
- 定期清理历史轨迹数据，防止内存溢出。
